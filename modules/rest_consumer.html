Package rest-consumer

@cordis/rest-consumer

About

A consumer/client-side for cordis' REST service

Getting started

Requirements
  • A rest-service instance hooked into the same RabbitMQ installation
  • Obviously, a RabbitMQ instance

Examples

 
const { Rest } = require('@cordis/rest-consumer');
const RedisClient = require('ioredis');

const main = async () => {
  const gateway = new Rest();

  const someUser = await rest.get('/users/223703707118731264');
  const someOtherGuy = await rest.make({
    path: '/users/198536269586890752',
    method: 'get'
  });

  console.log(someUser, someOtherGuy);
};

main();
