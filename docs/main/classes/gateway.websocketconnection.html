connect

  • connect(): Promise<void>
    • Defined in libs/gateway/src/websocket/WebsocketConnection.ts:313

    Connects to the gateway; resolves when: a) All guilds are recieved from Discord b) The timeout that waits for guilds is hit & the rest of the pending guilds are deemed unavailable. It should be noted that in either case when this function resolves this shard's status becomes 6 (ready)

    Returns Promise<void>
